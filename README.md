# TIFF_tags_nightly

Note: this version of TIFF_tags uses the Once_Cell feature from the nightly rust compiler 
in lieu of the Lazy_Static! macro.


A  lookup table to return the name of a TIFF/EXIF tag given its u16 tag ID.
